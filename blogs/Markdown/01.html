    我来学学学 Markdown 了

    LooseLi 2021/8/13 Markdown

    记录一下在学 Markdown 时遇到的各种语法，未必大全，但目前对笔者也够用了。后续学习到新的语法知识再回来补充

    # 标题

    共有一到六级标题，按级别依次增加一个#号

    格式：
    #一级标题内容
    ##二级标题内容
    ###三级标题内容
    ......

    # 换行符

    同 html 中的<br />标签

    # 加粗

    在内容前后分别用两个*号包裹

    格式：**需加粗内容**

    效果：哎嘿，加粗了

    # 斜体

    在内容前后分别用一个*号包裹

    格式：*需斜体内容*

    效果：我斜了

    tips：加粗斜体内容前后用三个*号包裹

    # 列表

    列表分有序列表和无序列表

    有序列表：数字 + . + 空格 + 内容

    格式：1. 有序列表

    1. 标题
    2. 换行符
    3. 列表

    无序列表：- + 空格 + 内容

    格式：- 无序列表

    • 标题
    • 换行符
    • 列表

    tips：无序列表的前缀除了可以用-，也可以用+或者*

    # 代码块

    分单行和多行代码块

    单行：将内容用``包裹

    效果：代码块

    多行：将内容用```包裹，效果如下：

    function(){
  console.log('Hi，老盐菠萝水');
};

    # 删除线

    在内容前后用~~包裹(英文状态下)

    格式：~~需删除内容~~

    效果：删了我

    # 分割线

    # 高亮
