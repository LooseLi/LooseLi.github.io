    push到GitHub时，提示403错误

    LooseLi 2021/8/16 GitGitHub

    作为万千打工人中的一员，笔者也是在今天准备开启一周的摸鱼时光 😏，那就继续来捣鼓博客吧。今天就来写个刚学习不久的 Vue3 吧！咦，push 时提示错误 ，作为摸鱼人，绝不能认输！在搜索了好久后，终于解决了。。。特来记录一下

    # 分析原因

    我们先来看一下 push 时提示的错误：

    上图大致意思是：从 2021 年 8 月 13 日起，基于密码的身份验证被弃用，请使用个人访问令牌

    基于此，我们就需要创建 personal access token，并更新我们的密码

    # 解决方法

    1. 创建 Personal access token

    在我们 github 账号的 Settings -> Developer settings -> Personal access tokens 下，创建一个新的 token，附上笔者的配置：

    点击创建即可。这时就会生成个人令牌。切记！！！，一定要立马把这个令牌复制并保存起来，因为一旦你关闭这个页面，这个令牌也就看不见了！

    2. 更新密钥链中的 GitHub 信息

    在终端输入Keychain access并启动程序，在该程序中搜索 github.com，并找到互联网密码这一条，双击打开，如图：

    勾选图中的显示密码，会让你输入密码，输入之后就会显示你的 github 登录密码。现在，用刚才创建的 token 替换这边的密码，并存储更改

    经过上面的步骤，我们就可以开心的 push 了～
